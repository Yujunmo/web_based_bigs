# web_based_bigs

세팅 및 실행 방법

1. 파이썬 가상환경 생성
2. 가상환경에 파일 다운로드
3. 가상환경 진입 (window : activate ,  linux,max : source bin/activate )
4. pip install -r requirements.txt
5. streamlit run login.py
