펀드 기간 수익률 로직 : (기준끝일의 펀드 수정기준가 - 기준시작일의 펀드 수정기준가) / 기준시작일의 펀드수정기준가 * 100  
BM 수익률 : (기준끝일의 BM기준가 - 기준시작일의 BM기준가) / 기준시작일의 BM기준가 * 100
펀드 연환산 수익률  로직 : (1 + 펀드기간 수익률 )^(1/운용일수) - 1
기하수익률 :(기간수익률 +1 )^(365/기간)-1
산술수익률 : (기간수익률)/기간 * 365
Sharpe (샤프 지수) : (펀드의 평균 수익률 - 무위험수익률평균) / 펀드의 표준편차 * sqrt(연율화일수)
Jensen(젠센 지수) : [ (펀드의 평균 수익률 - 무위험수익률평균) - 포트폴리오베타 * (BM평균수익률 - 무위험수익률평균) ]  * 연율화일수
Treynor(트레이너 지수) : (펀드의 평균수익률 - 무위험수익률평균)  / 포트폴리오베타 * 연율화일수 
Sharpe, Jensen, Treynor지수는 설정 후 31영업일 이후부터 조회가능)
기준가 = 당일NAV / 좌수 * 1000


수정기준가 = 기준가 * 누적분배율
누적분배율 = 펀드의 누적분배율은 펀드가 설정된 이후 지금까지 지급한 모든 분배금을 합산하여 이를 펀드 순자산가치(NAV) 대비 비율로 나타낸 것입니다.
